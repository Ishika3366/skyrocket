# Sky Rocket Venture

A Next.js website for e-commerce consulting services, featuring content from skyrocketventure.in, animations inspired by emizentech.com, e-commerce solutions from arvian.in, and a contact page like arvian.in.

## Features

- Responsive design with Tailwind CSS
- Smooth animations using Framer Motion
- Services for Amazon, Flipkart, Meesho account management
- Contact form and information
- Platform listings and client testimonials

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build the project:

```bash
npm run build
```

## Deploy

Deploy on Vercel or any hosting platform that supports Next.js.
